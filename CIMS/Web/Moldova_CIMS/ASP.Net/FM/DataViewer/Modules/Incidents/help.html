How to Install ‘WebAssembler Incident Screenshot Maker’ Extension in Google Chrome

  1. First of all download the extension from this website.

    To Download the extension click the link Download WebAssembler Incident Screenshot Maker

  2. In the Chrome browser, open a new tab by pressing Ctrl + T hotkey.

    In the address bar of this new tab, type and press Enter.

    This would open the Extensions page for your Google Chrome browser. You should un-maximize the chrome window – this is needed for the next step.
  3. Locate the downloaded Chrome extension file (.crx file) in the Windows Explorer. Then drag and drop it on the Extensions tab in the Google Chrome as shown in the picture below.

  4. You would now see the usual confirmation dialog for installing the extension. Click on the Add button to complete the installation.
  5. That’s it. You have successfully installed a Chrome extension.
