Form Object JScript Model

 

  Field's functions:

         getValue(name) - get value for the field

         getNumber(name) - get typed value

         getDate(name) - get typed value

         getLookupItems(name) - get typed lookup list {Value,Type,Text}

         getText(name) - get text value

 

         setValue(name, val) - set value for the field

         resetValue(name) - set previous value

 

         getLookupInfo(name, columns) - retrieve additional columns for selected lookup record

         loadInfo(serviceName, input, callback) - retrieve additional from the service more...

        

         RaiseOnChange(name) - fire field's ONCHANGE event

                  

         setRequired(name, b) - set/unset field's requirement state

         focusField(name) - focus field

         notifyField(name, s) - show popup message for the field

 

  Form's functions

         alert(hdr, msg) - show alert window

         toggleSection(section_name, b) - show/hide form's section

         getKeyValue() - return record ID value

         usePopupForRequired() - show Alert popups for Required Fields

         clickButton(cmd) - perform button click 'save', 'close', 'reload' and others.

        

  JQuery support

         getElement(id) - find JQuery html element in the form's container

 

  Example:

         frm.alert('Value for the field Description is: ' + frm.getValue('Description'));
